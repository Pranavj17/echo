# ECHO Documentation

This directory contains organized documentation for the ECHO project.

## Directory Structure

### `/guides/` - User Guides
Setup instructions, tutorials, and how-to guides for users and developers.

- **getting-started.md** - Initial setup and first steps
- **claude-desktop-setup.md** - Complete Claude Desktop integration guide
- **demo-guide.md** - Demo scenarios and examples

### `/architecture/` - Architecture Documentation
System design, technical architecture, and architectural decision records.

- **echo-architecture.md** - Complete system architecture overview

### `/troubleshooting/` - Troubleshooting & Fixes
Common issues, debugging guides, and fix documentation.

- **elixirls-connection-issue-explained.md** - ElixirLS debugging
- **db-id-fix-summary.md** - Database ID fix documentation

### `/completed/` - Completed Tasks
Historical logs of completed features, migrations, and major tasks.

- **day2-training-complete.md** - Training completion log
- **distributed-context-complete.md** - Context refactoring log
- **add-all-agents.md** - Agent addition tracking
- **cleanup-list.md** - Cleanup task tracking
- **agent-build-test-results.md** - Build test results
